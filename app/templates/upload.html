Options de traitement

Reconnaissance de texte (OCR)
L'OCR (Reconnaissance Optique de Caractères) lit le texte contenu dans le scan PDF (comme le nom de l'élève) pour permettre d'associer le bon bulletin au bon e-mail. Cochez "Forcer l'OCR" si les noms ne sont pas reconnus.

Ce texte constituera le corps de l'e-mail dans les brouillons .eml générés.

📮 Comment utiliser les fichiers .eml générés ?

  1. Téléchargez et décompressez l’archive finale .zip.
  2. Ouvrez le dossier eml/ (vous y trouverez un fichier par élève).
  3. Importez dans votre client mail : double-cliquez (Outlook/Mail) ou glissez dans vos Brouillons (Thunderbird).
  4. Vérifiez et envoyez !
