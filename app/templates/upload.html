📨 Comment utiliser les fichiers .eml générés

Les fichiers .eml sont des brouillons de mails prêts à être envoyés : destinataires (parents), objet, corps du message et PDF personnalisé.

  1. Téléchargez l’archive .zip en fin de traitement.
  2. Décompressez l’archive : ouvrez le dossier eml/ (un fichier par élève).
  3. Importez dans votre messagerie :
    • Thunderbird : glissez-déposez les .eml dans le dossier Brouillons.
    • Outlook / Mail : double-cliquez ou utilisez Ouvrir avec votre client mail.
  4. Relisez et envoyez (vous pouvez modifier le texte si besoin).
💡 Astuce : si le fichier ne s’ouvre pas, clic droit → Ouvrir avec → votre application de messagerie.
